# Task 2 – Mobile App UI for Appointment Booking

Business: GlowUp Beauty Salon

## Screens Designed

1. Onboarding Screen
2. Service Selection Screen
3. Date & Time Booking Screen
4. Appointment Confirmation Screen

## UX Design Rationale

The mobile app UI is designed to simplify the appointment booking process for users. The onboarding screen introduces the app and encourages users to get started quickly. The service selection screen presents available beauty services using clear visual cards for easy browsing.

The booking screen allows users to select a convenient date and time using a simple calendar and time-slot interface. Finally, the confirmation screen reassures users that their appointment has been successfully booked by displaying clear appointment details.

The overall UX focuses on clarity, speed, and a smooth booking flow to reduce friction and improve the user experience for service-based businesses.
